## ✦ Features

- **Add tasks** — Instantly add tasks via button click or `Enter` key
- **Complete tasks** — Toggle tasks as done with a satisfying checkbox
- **Delete tasks** — Remove individual tasks with smooth slide-out animation
- **Filter tasks** — Switch between All, Pending, and Completed views
- **Live stats** — Real-time count of total, done, and pending tasks
- **Clear completed** — Bulk-remove all finished tasks in one click
- **Persistent storage** — Tasks survive page reloads via `localStorage`
- **Date display** — Current date shown in the header
- **Fully responsive** — Looks great on mobile and desktop

---

## ✦ Usage

| Action | How |
|--------|-----|
| Add task | Type in input → click **Add Task** or press `Enter` |
| Complete task | Click the checkbox on the left of any task |
| Delete task | Hover over task → click the **×** button |
| Filter tasks | Click **All / Pending / Completed** tabs |
| Clear done | Click **Clear completed** at the bottom |

## ✦ Design

Tasky uses a **dark editorial aesthetic** with:
- Lime-green (`#c8f548`) accent on deep-dark backgrounds

- DM Sans for clean, readable body text

- Smooth micro-interactions on every element
